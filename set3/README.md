*	其中``GetRandomStr``函数的主要功能为随机选取字符串，并对这个字符串进行padding，最终对padding结果使用AES的CBC模式加密，并按照题目要求返回msg和iv值
*	其中``DecryptCheckPadding``函数的主要功能为对密文进行解密，并且检验padding的方式是否正确，检验padding的方式主要由上文中提到的``pkcs7unpad``函数提供。
